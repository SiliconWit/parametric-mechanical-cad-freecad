<BionicText method="advanced" intensity="medium" class="highlight">
The Geneva mechanism (also called Maltese cross mechanism) provides precise, intermittent rotary motion. The driven wheel indexes to exact positions while the driver rotates continuously, with self-locking dwell periods between each precise angular step.
</BionicText>

### Real-World Applications

The Geneva mechanism appears everywhere precise indexing is needed:

:::note[Common Applications]
- **Film projectors** - Advancing film frame-by-frame with precise positioning
- **Movie cameras** - Frame advance between exposures
- **Automatic machine tool changers** - Rotating tool turrets to exact positions
- **Conveyor indexing systems** - Precise product positioning
- **Rotary tables in manufacturing** - Exact angular positioning for multi-station work
- **Turret lathes** - Tool position indexing for automated operations
- **Packaging machinery** - Precise container positioning for filling operations
:::

### The Engineering Problem

> **Design Challenge:** Given continuous rotational input, how do we create intermittent, precise angular output with exact dwell periods and self-locking to prevent drift between indexes?

## üìö Mechanism Fundamentals

### Components and Motion

A Geneva mechanism consists of three key elements working together:

<Tabs>
  <TabItem label="Components">

    **1. Driver (Crank)**
    - Rotates continuously
    - Has a pin protruding from it
    - Includes locking arc on periphery

    **2. Driven Wheel (Geneva Wheel)**
    - Has radial slots (typically 4, 5, 6, or 8)
    - Indexes when pin engages slot
    - Remains stationary during dwell

    **3. Locking Arc**
    - Circular arc on driver
    - Engages driven wheel's periphery
    - Prevents rotation during dwell phase

</TabItem>
  <TabItem label="Operation Phases">

    **Phase 1: Dwell (Locked Position)**
    - Driver's locking arc engages driven wheel's periphery
    - Driven wheel prevented from rotating
    - No motion despite driver rotation

    **Phase 2: Index (Motion)**
    - Driver pin enters driven wheel slot
    - Driven wheel rotates to next position
    - Precise angular movement occurs

    **Phase 3: Return to Dwell**
    - Pin exits slot
    - Locking arc re-engages
    - Driven wheel held stationary again

    :::tip[Engineering Insight]
    The locking arc is critical! Without it, the driven wheel could drift or vibrate during dwell, destroying precision. This self-locking feature is what makes Geneva mechanisms superior to simple indexing cams.
    :::

</TabItem>
  <TabItem label="Key Parameters">

    <Card title="‚öôÔ∏è Critical Design Parameters" icon="setting">

    **Number of slots (n)**
    - Typically 4, 5, 6, or 8
    - Determines index angle: Œ∏ = 360¬∞ / n
    - More slots = smaller index steps

    **Driver radius (R_d)**
    - Distance from driver center to pin center
    - Controls mechanism size and geometry

    **Driven wheel radius (R_g)**
    - Distance from center to slot entry
    - Calculated: R_g = R_d √ó tan(180¬∞ / n)

    **Dwell ratio**
    - Fraction of driver rotation where driven wheel is stationary
    - For n slots: Dwell = (n - 2) / n of full rotation

    </Card>

</TabItem>
</Tabs>

### Mathematical Relationships

For an external Geneva mechanism with n slots:

<Tabs>
  <TabItem label="Geometric Formula">

    **Driven wheel radius calculation:**
    ```
    R_g = R_d √ó tan(180¬∞ / n)
    ```

    **Where:**
    - R_g = Geneva wheel radius (center to slot entry)
    - R_d = Driver radius (center to pin)
    - n = Number of slots

    **Slot spacing:**
    ```
    Slot center angle = 360¬∞ / n
    ```

</TabItem>
  <TabItem label="Timing Analysis">

    **Index angle per step:**
    ```
    Œ∏ = 360¬∞ / n
    ```

    **Dwell angle:**
    ```
    Dwell = 360¬∞ √ó (n - 2) / n
    ```

    **Index duration:**
    ```
    Index = 360¬∞ / n
    ```

    :::note[Example: 6-Slot Geneva]
    - Index angle: 360¬∞ / 6 = 60¬∞ per step
    - Dwell: 360¬∞ √ó (4/6) = 240¬∞ of driver rotation
    - Index: 360¬∞ / 6 = 60¬∞ of driver rotation
    - The driven wheel is stationary for 2/3 of each cycle!
    :::

</TabItem>
  <TabItem label="Common Configurations">

    | Slots | Index Angle | Dwell Ratio | Application |
    |-------|-------------|-------------|-------------|
    | **4-slot** | 90¬∞ | 50% (180¬∞) | Film projectors |
    | **6-slot** | 60¬∞ | 67% (240¬∞) | Machine tool changers |
    | **8-slot** | 45¬∞ | 75% (270¬∞) | High-frequency indexing |

    We'll build a **6-slot Geneva** for balanced performance and clear learning.

</TabItem>
</Tabs>

## üí° Part 2: Parametric Design Strategy

<BionicText method="advanced" intensity="medium" class="highlight">
Geneva mechanisms are parameter-driven by nature. Change the number of slots, and everything else must recalculate geometrically. This makes them perfect for practicing intelligent parametric design where formulas and relationships drive the entire model.
</BionicText>

### Our Design Approach

<Card title="üéØ Multi-Parameter Control Philosophy" icon="rocket">

We'll control the **entire Geneva mechanism** with key parameters:

**NumSlots** = 6 (number of slots)
**DriverCenterDistance** = 80 mm (between wheel centers)
**PinDiameter** = 10 mm

From these, FreeCAD will **automatically calculate**:
- Slot angle spacing (60¬∞)
- Driven wheel radius (geometric formula)
- Index angle per step
- Slot width (with clearance)

This is intelligent parametric design!

</Card>

### Our Geneva Configuration

We'll design a **6-slot external Geneva mechanism**:

<Tabs>
  <TabItem label="Configuration">

    **Design Specifications:**
    - **6 radial slots** on driven wheel
    - **Index angle:** 360¬∞ / 6 = 60¬∞ per step
    - **Driver:** Rotates continuously
    - **Dwell period:** Driver completes 240¬∞ (4/6 of rotation)
    - **Index period:** Driven wheel rotates 60¬∞ while driver completes 120¬∞ (2/6 of rotation)

</TabItem>
  <TabItem label="Design Workflow">

    **Build Sequence:**
    ```
    Step 1: Create Spreadsheet (with formulas!)
    Step 2: Create Driver Wheel (with pin)
    Step 3: Create Geneva Wheel (with polar slot pattern)
    Step 4: Create Frame (mounting base)
    Step 5: Assemble with rotation constraints
    Step 6: Create technical drawing with timing diagram
    Step 7: Test parametric control (change slot count!)
    ```

</TabItem>
  <TabItem label="Why This Configuration?">

    :::note[6-Slot Selection Rationale]
    **Why 6 slots?**
    - **Balanced geometry** - 60¬∞ angles are easy to visualize and verify
    - **Good dwell ratio** - 67% of cycle is stationary (ideal for many applications)
    - **Not too complex** - Clear patterns without overwhelming detail
    - **Real-world relevance** - Common in machine tool indexers
    - **Easy to modify** - Change to 4 or 8 slots to see principles apply universally
    :::

    </Steps>

</TabItem>
  <TabItem label="First Slot">

    **Draw one radial slot (we'll pattern it later):**

    This is the tricky part! We need a radial slot.

    <Steps>

    1. **Rectangle tool** (or draw with lines)
    2. Draw a rectangle extending **radially outward** from near the center
    3. Position it roughly on the **horizontal axis to the right**
    4. Press **Escape**

    5. **Make slot symmetric about horizontal axis:**
       - Select **Symmetric constraint** tool
       - Click rectangle **top edge**
       - Click rectangle **bottom edge**
       - Click **horizontal axis** (X-axis)
       - This centers the slot!

    6. **Slot width (perpendicular to radius):**
       - **Distance constraint**
       - Click top edge
       - Click bottom edge
       - Click **∆íx** button
       - Type: `Spreadsheet.SlotWidth`
       - Press **Enter**

    7. **Slot position - inner end:**
       - The slot should start close to the shaft hole
       - **Distance constraint** from origin to slot inner end
       - Type: `25` mm (or just inside the wheel radius)

    8. **Slot position - outer end:**
       - The slot should extend to or beyond the wheel periphery
       - **Distance constraint** from origin to slot outer end
       - Type: equal to or slightly beyond `Spreadsheet.DrivenRadius`
       - Or simply extend past the circle (will be trimmed by wheel boundary)

    </Steps>

    :::tip[Slot Geometry Tips]
    - Slot must be **radial** (pointing toward center)
    - Slot must **extend beyond** wheel periphery to ensure clean cutout
    - Width controlled by `SlotWidth` parameter (has clearance for pin)
    :::

</TabItem>

## Mathematical Background

### Geneva Wheel Radius Calculation

For an external Geneva mechanism:

```
R_g = R_d √ó tan(180¬∞ / n)

Where:
R_g = Geneva wheel radius (center to slot entry)
R_d = Driver radius (center to pin)
n = Number of slots
```

**For n = 6:**

```
R_g = R_d √ó tan(30¬∞)
R_g = R_d √ó 0.577
```

If R_d = 40mm, then R_g ‚âà 23mm.

### Dwell vs. Index Time

**Dwell angle:** 360¬∞ √ó (n - 2) / n

**Index angle:** 360¬∞ / n

**For 6-slot Geneva:**
- Dwell: 360¬∞ √ó 4/6 = 240¬∞ of driver rotation
- Index: 360¬∞ √ó 1/6 = 60¬∞ of driver rotation

## Common Issues and Solutions

### "Slots not evenly spaced"

- **Cause:** Polar pattern not set to correct angle or count
- **Solution:** Verify SlotAngle = 360 / NumSlots
- **Fix:** Check pattern constraints reference spreadsheet parameters

### "Pin doesn't fit in slot"

- **Cause:** Slot width too narrow
- **Solution:** SlotWidth should be 1.2-1.5 √ó PinDiameter
- **Fix:** Adjust SlotWidth parameter

### "Geneva wheel radius seems wrong"

- **Cause:** Incorrect TAN formula or units
- **Solution:** Verify formula: =B8*TAN(180¬∞/B2)
- **Fix:** Check that TAN function uses degrees (or convert with RADIANS())

### "Mechanism binds during motion"

- **Cause:** Center distance incorrect or slot angle wrong
- **Solution:** Recalculate center distance using correct geometry
- **Fix:** Verify all geometric relationships
